IF文

IF文は条件分岐の際に使用します。

条件のところは～ならと書きます。(例:A<B)←AよりBが大きければ

「条件が成り立った場合処理を実行」のところは条件に該当した場合どんな処理をするかを書きます。

(例:"Hello")条件に該当した場合Helloと表示されます。

「else if」は2つ目の条件

「else」は条件に該当しなかった場合の処理

「else」を書かないと条件に該当しない場合は何も処理を実行しない事になる。

                        
                            if (条件1) {
                                条件1が成り立った場合処理を実行
                            } else if (条件2) {
                                条件2が成り立った場合処理を実行
                            } else {
                                条件1と条件2が成り立たなかった場合処理を実行
                            }
